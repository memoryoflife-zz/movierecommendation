This agent is divided into four parts:
1. Buying flight tickets
2. Buying hotel vouchers
3. Buying entertainment tickets
4. Selling entertainment tickets

Each part of the could is divided by the comment line

At the end of the file there are two generic method called getMarket and getBedmarket, which is 
used inside the runagent method in order to provide a convenient way to have access to these markets

All the main part of the code is in the runagent method, which is used to perform agent tasks.

More instructions could be seen in the inline comment.
